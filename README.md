# 简介
Porter是一个将视频从抖音搬运到全民小视频的项目,前端为Vue+Element,后端为Golang,项目中主要实现了以下功能:

1. 实现了抖音X-Gorgon加密算法；
2. 按关键字添加关注的抖音账户;
3. 固定时间搜索所有关注的抖音账户是否有新视频更新;
4. 无水印视频下载并上传到全民小视频;
5. 将抖音用户信息自动同步到指定的全民小视频账号;

个人研究型项目,有些地方的代码比较粗糙请见谅,欢迎提交PR.
